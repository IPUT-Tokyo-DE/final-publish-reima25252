# dodger

## ゲームの内容
３０秒間の弾避け。ジャンプとブリンクで高密度の弾幕から生き残る

![画面イメージ](docs/images/game_image01.png)

## 操作説明
- 矢印キー、WASDで移動、shiftでブリンク、スペースでジャンプ
